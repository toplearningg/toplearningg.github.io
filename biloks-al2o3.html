5. biloks dari 2nah2po3 ?biloks dari 2ph3 ?​


Jawaban:

Biloks dari 2NaH2PO3 (natrium hipofosfit) adalah:

- Biloks natrium (Na) adalah +1 karena ia adalah logam alkali.

- Biloks hidrogen (H) adalah +1 karena ia umumnya memiliki bilangan oksidasi positif ketika terikat dengan nonlogam.

- Biloks fosfor (P) adalah +1 karena ada dua hidrogen yang terikat ke fosfor dan masing-masing hidrogen memiliki bilangan oksidasi +1. Namun, ada dua atom oksigen di sebelah fosfor, yang masing-masing memiliki bilangan oksidasi -2. Karena total muatan dari senyawa harus netral, maka bilangan oksidasi dari fosfor harus +1 untuk menetralkan total bilangan oksidasi hidrogen dan oksigen. Oleh karena itu, bilangan oksidasi fosfor dalam 2NaH2PO3 adalah +1.

Jadi, biloks dari 2NaH2PO3 adalah: Na(+1) + 2H(+1) + P(+1) + 3O(-2) = 0.

Sedangkan, biloks dari 2PH3 (difosfina) adalah:

Biloks fosfor (P) adalah -3 karena ia terikat dengan tiga atom hidrogen (H), dan masing-masing hidrogen memiliki bilangan oksidasi +1. Oleh karena itu, untuk menetralkan total bilangan oksidasi hidrogen dan fosfor harus -3.

Jadi, biloks dari 2PH3 adalah: 2P(-3) + 6H(+1) = 0.

7. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4

27. 1. Biloks Mn pada K2MnO4 2. Biloks N pada NH4+ 3. Biloks Cr pada K2CrO7 4. Biloks O pada Na2O2 5. BILOKS Mn pada KMnO4 6. Biloks S pada Na2SO4 7. Biloks H pada Na2H2


1. Biloks Mn pada K₂MnO₄ adalah +6.

2. Biloks N pada NH₄⁺ adalah -3.

3. Biloks Cr pada K₂CrO₇ adalah +12.

4. Biloks O pada Na₂O₂ adalah -1.

5. Biloks Mn pada KMnO₄ adalah +7.

6. Biloks S pada Na₂SO₄ adalah +6.

7. Biloks H pada Na₂H₂ adalah -1.

Pembahasan : 1. Biloks Mn pada K₂MnO₄

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂MnO₄ = 0  

(2 × biloks K) + biloks Mn + (4 × biloks O) = 0  

(2 × (+1)) + biloks Mn + (4 × (-2)) = 0  

(+2) + biloks Mn + (-8) = 0  

biloks Mn + (-6) = 0  

biloks Mn = +6

2. Biloks N pada NH₄⁺

Biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H) = +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N = (+1) - 4

biloks N = -3

3. Biloks Cr pada K₂CrO₇

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0  

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0  

(+2) + biloks Cr + (-14) = 0  

biloks Cr + (-12) = 0  

biloks Cr = +12

4. Biloks O pada Na₂O₂

biloks Na = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks Na₂O₂ = 0  

(2 × biloks Na) + (2 × biloks O) = 0  

(2 × (+1)) + (2 × biloks O) = 0  

(+2) + (2 × biloks O) = 0  

(2 × biloks O) = -2

biloks O = [tex]\frac{-2}{2}[/tex]

biloks O = -1

5. Biloks Mn pada KMnO₄

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks KMnO₄ = 0  

biloks K + biloks Mn + (4 × biloks O) = 0  

(+1) + biloks Mn + (4 × (-2)) = 0  

(+1) + biloks Mn + (-8) = 0  

biloks Mn + (-7) = 0  

biloks Mn = +7

6. Biloks S pada Na₂SO₄

biloks Na = +1 (merupakan unsur golongan IA)

aturan biloks :

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2

aturan biloks :

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks Na₂SO₄ = 0  

(2 × biloks Na) + biloks S + (4 × biloks O) = 0  

(2 × (+1)) + biloks S + (4 × (-2)) = 0  

(+2) + biloks S + (-8) = 0  

biloks S + (-6) = 0  

biloks S = +6

7. Biloks H pada Na₂H₂

biloks Na = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks Na₂H₂ = 0  

(2 × biloks Na) + (2 × biloks H) = 0  

(2 × (+1)) + (2 × biloks H) = 0  

(+2) + (2 × biloks H) = 0  

(2 × biloks H) = -2

biloks H = [tex]\frac{-2}{2}[/tex]

biloks H = -1

29. Tentukan biloks dari umsur: Tentukan biloks Mg dari Mg3(PO4)2 biloks Cu dari CuCl2 dan biloks C dari CaCO3 Tolong dijawab ya


Biloks dari unsur berikut :  

1.  Biloks Mg dari Mg₃(PO₄)₂ adalah +2.

2.  Biloks Cu dari CuCl₂ adalah +2.

3.  Biloks C dari CaCO₃ adalah +4.

Pembahasan :1. Biloks Mg dari Mg₃(PO₄)₂

biloks P = +5

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.  

biloks Mg₃(PO₄)₂ = 0  

(3 × biloks Mg) + (2 × biloks P) + (8 × biloks O) = 0  

(3 × biloks Mg) + (2 × (+5)) + (8 × (-2)) = 0  

(3 × biloks Mg) + (+10) + (-16) = 0  

(3 × biloks Mg) + (-6) = 0  

(3 × biloks Mg) = +6

Biloks Mg = [tex]\frac{+6}{3}[/tex]

Biloks Mg = +2

2. Biloks Cu dari CuCl₂

biloks Cl = -1 (Golongan VIIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks CuCl₂ = 0  

biloks Cu + (2 × biloks Cl) = 0  

biloks Cu + (2 × (-1)) = 0  

biloks Cu + (-2) = 0  

biloks Cu = +2

3. Biloks C dari CaCO₃

biloks Ca = +2 (merupakan unsur golongan IIA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaCO₃ = 0  

biloks Ca + biloks C + (3 × biloks O) = 0  

(+2) + biloks C + (3 × (-2)) = 0  

(+2) + biloks C + (-6) = 0  

biloks C + (-4) = 0  

biloks C = +4

⇒ Kesimpulan, maka biloks dari unsur berikut :  

1.  Biloks Mg dari Mg₃(PO₄)₂ adalah +2.

2.  Biloks Cu dari CuCl₂ adalah +2.

3.  Biloks C dari CaCO₃ adalah +4.

Penentuan Bilangan Oksidasi

Bilangan oksidasi atau tingkat oksidasi suatu unsur adalah bilangan yang menyatakan jumlah muatan yang disumbangkan oleh atom atau unsur tersebut pada molekul atau ion yang dibentuknya.
