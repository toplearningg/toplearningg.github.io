1. Jika besar sudut AOB 4x besar sudut BOC, maka besar sudut AOB sebenarnya adalah .​


Jawaban:

Sudut AOB = 144 derajat

Penjelasan dengan langkah-langkah:

Sudut AOB = 4x

Sudut BOC = x

Sudut AOC = 180 derajat (karena garis lurus)

Sudut AOC = Sudut AOB + Sudut BOC

180 = 4x + x

180 = 5x

180/5 = x

x = 36 derajat

Sudut AOB = 4x = 4(36) = 144 derajat

12. besar sudut AOB adalah​


Jawaban:

Besar sudut AOB adalah 108°

Pendahuluan :

Sudut AOB saling berpelurus dengan sudut BOC

Sudut berpelurus adalah dua buah sudut yang membentuk garis lurus, dan besar sudut berpelurus adalah 180°

Pembahasan :

Diketahui :

[tex] \angle\: aob = 3x[/tex]

[tex] \angle \: boc = 2x[/tex]

Ditanya :

Besar sudut AOB...?

Jawab :

>> cari nilai x

[tex] \angle \: aob + \angle \: boc = {180}^{0} [/tex]

[tex]3x \: + \: 2x = {180}^{0} [/tex]

[tex]5x \: = {180}^{0} [/tex]

[tex]x = \frac{180}{5} [/tex]

[tex]x = {36}^{0} [/tex]

>> menentukan besar sudut AOB

[tex] \angle \: aob = 3x[/tex]

[tex] \angle \: aob = 3 \times 36[/tex]

[tex] \angle \: aob = {108}^{0} [/tex]

Kesimpulan :

Jadi besar sudut AOB adalah 108°

=> Sudut berpenyiku = 90°

A. Besar nilai X

• 4x° + 2x° = 90°

• 6x° = 90°

• x = 90° / 6

• x = 15°

B. Besar penyiku <AOB = <BOC

• <BOC = 2x°

• <BOC = 2(15°)

• <BOC = 30°
