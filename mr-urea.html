24. Hitung fraksi mol urea dalam larutan urea yang mengandung 30 gram urea da 81 gram air, jika Mr urea =60 dan Mr air =18?


Fraksi mol urea berarti fraksi mol terlarut. Rumus:
X = n terlarut / (n terlarut + n pelarut)

Diket:
g(massa) terlarut= 30 gr
g(massa) pelarut= 81 gr
Mr terlarut = 60
Mr pelarut = 18
maka
n ter = g ter / Mr ter
= 30 / 60
= 0,5 mol

n pel = g pel / Mr pel
= 81 / 18
= 4,5 mol

X = n terlarut / (n terlarut + n pelarut)
= 0,5 / (0,5 + 4,5)
= 0,5 / 5
= 0,1

Maka fraksi mol urea adalah 0,1
