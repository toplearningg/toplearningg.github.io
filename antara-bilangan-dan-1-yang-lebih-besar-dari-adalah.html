1. nilai rata-rata dari 6 bilangan adalah 56 selisih bilangan terbesar dengan bilangan terkecil 30 ada 1 bilangan terbesar dan bilangan lainnya sama besar bilangan terbesar adalah​


anggap aja urutan bilangan dari terkecil ke terbesar A,B,C,D,E,F..

berarti

A + B + C + D + E + F = 56(6)

A + B + C + D + E + F = 336

1 bilangan terbesar = F

bilangan lain sama besar = A= B = C = D = E

F - A = 30 --> A = F - 30

F-B = 30 -> B = F - 30

F - C = 30 --> C = F -30

F - D = 30 --> D = F - 30

F - E = 30 --> E = F - 30

jadi

(F-30) + (F-30)+ (F-30) + (F-30) + (F-30)+ F = 336

6F - 150 = 336

6F = 486

F = 81

bilangana terbesar ya berarti = 81

10. Tiga bilangan bulat jika dijumlahkan hasilnya adalah 8.Bilangan paling besar dan paling kecil jika dijumlahkan hasilnya adalah 3.Jika diurutkan ,selisih bilangan yang ada di tengah dan paling besar adalah 1.Tentukan selisih bilangan paling besar dan bilangan paling kecil! ​


Selisih bilangan paling besar dan bilangan paling kecil adalah 9. Simak penjelasan berikut.

Pembahasan

Diketahui

Jumlah tiga bilangan bulat adalah 8

Jumlah bilangan paling besar dan paling kecil adalah 3

Selisih bilangan yang tengah dan paling besar adalah 1

Ditanya

Selisih bilangan paling besar dan bilangan paling kecil

Penyelesaian

Misalkan,

a = bilangan bulat paling kecil

b = bilangan bulat yang ada tengah nilainya

c = bilangan bulat paling besar

Maka,

Jumlah tiga bilangan bulat adalah 8

a + b + c = 8 ... (persamaan 1)

Jumlah bilangan paling besar dan paling kecil adalah 3

a + c = 3 ... (persamaan 2)

Selisih bilangan yang tengah dan paling besar adalah 1

c - b = 1 ... (persamaan 3)

* Substitusikan nilai persamaan 2 ke persamaan 1

a + b + c = 8

a + c + b = 8

3 + b = 8

b = 8 - 3

b = 5

* Substitusikan nilai b ke persamaan 3

c - b = 1

c - 5 = 1

c = 5 + 1

c = 6

* Substitusikan nilai c ke persamaan 2

a + c = 3

a + 6 = 3

a = 3 - 6

a = -3

Selisih bilangan paling besar dan bilangan paling kecil

= c - a

= 6 - (-3)

= 9

Kesimpulan

Jadi, selisih bilangan paling besar dan bilangan paling kecil adalah 9.

❐ BILANGAN PECAHAN ✍

Bilangan pecahan adalah sebuah bilangan yang terdiri atas 2 angka yang berperan sebagai pembilang dan penyebut. Bilangan pecahan dapat dikelompokkan lagi menjadi 2, yaitu bilangan pecahan biasa dan bilangan pecahan campuran.

Bilangan pecahan biasa ✨

Bilangan pecahan biasa adalah bilangan yang hanya terdiri dari dua angka saja, yakni satu angka berperan sebagai pembilang dan angka lainnya sebagai penyebut yang masing-masing berada diatas dan dibawah. Pembilang berada diatas, sedangkan penyebut berada dibawah. Contoh bilangan pecahan biasa adalah sebagai berikut :

[tex] \frac{1}{2} \\ [/tex]

[tex] \frac{2}{6} \\ [/tex]

[tex] \frac{8}{12} \\ [/tex]

_______________________

Bilangan pecahan campuran ✨

Bilangan pecahan campuran adalah sebuah bilangan yang memiliki 3 angka yang berperan sebagai bilangan asli, pembilang dan penyebut. Bilangan asli ditulis dengan ukuran yang lebih besar di paling kiri, sedangkan pembilang berada diatas, dan penyebut berada di bawah. Bilangan pecahan campuran dapat dikatakan sebagai bentuk yang paling sederhana dalam suatu bilangan pecahan. Untuk memperoleh bilangan pecahan campuran, kita harus memiliki pembilang yang nilainya lebih besar dari penyebut, jika sebaliknya maka kita tidak dapat memperoleh bilangan pecahan campuran. Contoh bilangan pecahan campuran adalah sebagai berikut :

[tex]3 \frac{5}{8} \\ [/tex]

[tex]2 \frac{8}{16} \\ [/tex]

[tex]5 \frac{18}{24} \\ [/tex]

Tiga bilangan bulat jika dijumlahkan hasilnya adalah 8. Bilangan paling besar dan paling kecil jika dijumlahkan hasilnya adalah 3. Jika diurutkan, selisih bilangan yang ada di tengah dan paling besar adalah 1. Selisih bilangan paling besar dan bilangan paling kecil adalah 9

Kita akan mempelajari tentang aljabar dan  sistem persamaan linier 3 variabel

Aljabar adalah  salah satu cabang matematika yang mempelajari tentang pemecahan masalah menggunakan simbol-simbol sebagai pengganti konstanta dan variabel

Untuk menyelesaikan sistem persamaan linier 3 variabel kita dapat menggunakan cara :

EliminasiSubstitusiGrafikMatrix

Pembahasan :

Diketahui :

3 buah bilangan bulat

misalnya :

paling besar = a

bilangan tengah = b

paling kecil = c

a + b + c = 8.....pers (1)

a + c = 3.......pers(2)

a - b = 1......pers (3)

Ditanya :

Selisih bilangan paling besar dan paling kecil ?

Dijawab :

Ini merupakan Sistem persamaan linier 3 variabel, pertama-tama kita eliminasi pers 2 dan pers 3

a + c = 3

a - b = 1 -

b + c = 2.....pers (4)

Selanjutnya kita eliminasi pers 1 dan pers 4

a + b + c = 8

     b + c = 2 -

           a = 6

Kita substitusi pers 2 ke pers 1

a + c = 3

a + b + c = 8

(a + c) + b = 8

3 + b = 8

b = 8 - 3

b = 5

Lalu kita substitusi ke pers 1

a + b + c = 8

6 + 5 + c = 8

11 + c = 8

c = 8 - 11

c = -3

angka terbesar = 6

angka tengah = 5

angka terkecil = -3

Kita cari selisih bilangan paling besar dengan paling kecil

6 - (-3) =

6 + 3 =

9

∴ Jadi selisih bilangan paling besar dengan bilangan paling kecil adalah 9
