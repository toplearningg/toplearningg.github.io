Prisma

Bangun prisma adalah bangun ruang sisi datar yang memiliki alas dan tutup yang sama bentuknya.

Banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2

Banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3n

Banyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n

Pada prisma, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama prisma tersebut adalah prisma segitiga, begitu seterusnya.

Limas

Limas adalah bangun ruang yang alasnya berbentuk segi banyak, dan sisi tegaknya berbentuk segitiga.

Banyak sisi pada limas segi-n dapat dirumuskan sebagai n + 1

Banyak rusuk pada limas segi-n dapat dirumuskan sebagai 2n

Banyak titik sudut pada limas segi-n dapat dirimuskan sebagai n + 1

Pada limas, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama limas tersebut adalah limas segitiga, begitu seterusnya.

Pembahasan

1. Banyak nya rusuk dan sisi dari limas segi 7 berturut turut adalah...

Rusuk = 2n

          = 2.7

          = 14

Sisi = n + 1

      = 7 + 1

      = 8

2. Alas sebuah limas berbentuk segi 8. banyaknya rusuk dan sisi limas berturut- turut adl...

Rusuk = 2n

          = 2.8

          = 16

Sisi = n + 1

      = 8 + 1

      = 9

3. Sebuah prisma mempunyai alas berbentuk segi 10. banyaknya sisi dan rusuk pada prisma adl..

Sisi = n + 2

      = 10 + 2

      = 12

Rusuk = 3n

          = 3.10

          = 30

Banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2Banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3nBanyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n

Pada prisma, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama prisma tersebut adalah prisma segitiga, begitu seterusnya.

Limas

Limas adalah bangun ruang yang alasnya berbentuk segi banyak, dan sisi tegaknya berbentuk segitiga.

Banyak sisi pada limas segi-n dapat dirumuskan sebagai n + 1Banyak rusuk pada limas segi-n dapat dirumuskan sebagai 2nBanyak titik sudut pada limas segi-n dapat dirimuskan sebagai n + 1

Pada limas, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama limas tersebut adalah limas segitiga, begitu seterusnya.

Pembahasan

Prisma segi-8

Banyak sisi pada prisma segi-8  

=  8 + 2

= 10

Banyak rusuk pada prisma segi-8  

= 3 x 8

= 24

Banyak titik sudut pada prisma segi-8  

= 2 x 8

= 16

Limas segi-8

Banyak sisi pada limas segi-8

= 8 + 1

= 9

Banyak rusuk pada limas segi-8

= 2 x 8

= 16

Banyak titik sudut pada limas segi-8

=  8 + 1

= 8
