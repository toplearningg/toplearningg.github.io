4. apa perbedaan antara menghapus dan uninstall? berikan contohya


Uninstall yaitu proses yang akan menghapus beberapa file yang terkait dengan suatu aplikasi. Contohnya, melakukan uninstall Microsoft Office, tindakan uninstall akan menghapus shortcut, file-file, dan registry terkait program Microsoft Office, yang kemungkinan kurang di ketahui letaknya.

Sedangkan menghapus, contohnya menghapus folder Microsoft Office yang terletak pada folder Program Files, hanya akan menghapus folder tersebut, sedangkan shortcut, file-file, dan registry terkait program tersebut masih akan tersimpan.

11. Apakah yang akan terjadi kalau pengguna sering install dan uninstall software


Tidak akan terjadi masalah apapun jika user terus-menerus reinstall suatu software, tapi ada kemungkinan akan ada error. Namun, kemungkinan itu sangat kecil. Biasanya software akan membuat file cache.

File cache ini digunakan untuk menyimpan data yang nantinya akan digunakan untuk memulai kembali aplikasi. Jadi, aplikasi tidak perlu memuat data yang banyak karena sudah tersimpan pada cache.

Bisa saja terjadi error saat memulai kembali aplikasi yang terus-menerus di-reinstall. Misalnya, file cache tersebut menumpuk dan aplikasi terkena freeze atau force close. Hal yang dapat mencegahnya adalah selalu membersihkan file cache sebelum meng-uninstall suatu software.

27. Assalamualaikum, mau tanya dong.. ada yg ngerti ga gmn cara nya donwload aplikasi microsoft photo, soalnya ke uninstall, terus mau install di m store nya ga jalan gimana ya solusinya?​


Jawaban:

Metode 1

Uninstall aplikasi Photos melalui PowerShell

Langkah 1: Jalankan PowerShell sebagai administrator. Untuk melakukannya, ketikkan PowerShell pada kolom pencarian di Start menu atau taskbar, klik kanan pada entri PowerShell di hasil pencarian, kemudian klik opsi Run as administrator.

Klik tombol Yes jika Anda melihat kotak User Account Control.

Langkah 2: Pada PowerShell, ketikkan command berikut ini kemudian tekan tombol Enter untuk melihat aplikasi yang sudah diinstall.

Get-AppxPackage –AllUsers

Langkah 3: Gulir ke bawah, cari entri bernama Microsoft.Windows.Photos, kemudian copy PackageFullName ke program Notepad. Untuk bisa menemukan entri aplikasi Photos dengan cepat, klik kanan pada titlebar PowerShell, klik Edit -> Find, lalu ketikkan Photos. Setelah itu, klik tombol Find next.

Untuk menyalin PackageFullName, pilih PackageFullName kemudian gunakan hotkey Ctrl + C untuk menyalinnya.

Baca Juga:  5 Penanganan Pertama pada PC yang Rusak

Langkah 4: Jalankan command berikut ini untuk meng-uninstall aplikasi Photos.

remove-AppxPackage PackageFullName

Pada command di atas, ganti PackageFullName dengan PackageFullName aplikasi Photos yang sebelumnya sudah Anda salin ke notepad di langkah 3.

Langkah 5: Setelah meng-uninstall aplikasi Photos dari Windows 10, sekarang Anda bisa menginstall aplikasi Photos dari Windows Store. Saat Store terbuka, ketikkan Microsoft photos pada kolom pencarian, klik pada entri Microsoft Photos, kemudian klik tombol Free. Jika tombol Free tidak aktif, atau yang ada hanyalah tombol Open, tutup kembali Windows Store. Kemudian jalankan kembali Windows Store tersebut.

Metode 2

Menginstall ulang aplikasi Photos menggunakan CCleaner

Bagi Anda yang tidak familiar dengan PowerShell, Anda juga bisa menggunakan software gratis bernama CCleaner untuk menguninstall aplikasi foto. Berikut ini caranya.

Langkah 1: Unduh software CCleaner, kemudian install pada PC Windows 10 Anda.

Langkah 2: Jalankan CCleaner, klik Tools, lalu klik Uninstall untuk melihat daftar aplikasi yang telah terinstall.

Langkah 3: Cari entri dengan nama Photos, klik kanan pada entri tersebut, kemudian klik tombol Uninstall. Klik OK jika Anda melihat kotak konfirmasi.
